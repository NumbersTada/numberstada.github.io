How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

This is a modified version of 2048 that supports any grid size. Colors for nonexistent tiles are automatically taken from other defined tiles. If you want to make a modified 2048 game, you should use this version, as this version includes some improvements to the official 2048. Modification created by NumbersTada.
