How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

X tiles can't be merged with each other, making the game harder. If you enable the "Movable X tiles" checkbox, you can move the X tiles.
Important: Some boards may be impossible depending on the settings. The X tile generation is completely random, so you may end up with an impossible board if you select a high number of X tiles or disable movable X tiles.
