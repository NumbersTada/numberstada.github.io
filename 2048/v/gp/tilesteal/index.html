How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

Tiles disappear every half a second. Every time a tile disappears, the value of the tile is multiplied by 4 and this value is removed from your score. If the board is empty or you have negative score, you lose.

If this is too hard, you might want to try out Speed 2048.
